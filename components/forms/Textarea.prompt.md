**Input / Textarea** — calm, generous form fields with a label above and optional helper below; warm focus ring, large hit target.

```jsx
<Input label="Your email" placeholder="you@example.com" helper="We'll send your Starter Kit here. No spam." />
<Textarea label="The prompt to copy" monospace readOnly value={"Explain growing tomatoes…"} />
```

Set `monospace` + `readOnly` on Textarea for the read-only worked-example prompt box.
