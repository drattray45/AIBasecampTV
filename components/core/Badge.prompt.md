**Badge** — a small, calm uppercase label. Never alarmist.

```jsx
<Badge tone="free">Free</Badge>
<Badge tone="success">Nice — you did that</Badge>
<Badge tone="note">Reserved</Badge>
```

Tones: `neutral`, `free` (amber quiet wash — the hero kit), `success` (gentle leaf green), `note` (warm clay, for honesty/reserved notes).
