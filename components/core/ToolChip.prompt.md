**ToolChip** — a plain, honest name-tag for a familiar tool, used in the "tools you'll actually use" bar instead of fabricated customer logos.

```jsx
<ToolChip>ChatGPT</ToolChip>
<ToolChip>Claude</ToolChip>
<ToolChip>Canva</ToolChip>
```

Never render a real third-party logo inside it — plain type only, per the brand's "don't fake it" rule.
